# Redmine document view overwrite plugin
- 文書の一覧表示を変更するプラグインです。
    - 文書の並び順を降順にします。
    - 1文書あたりの表示が長すぎると感じるので、表示を切り詰めました。

## Instllation
- モジュールのコピー
```
$ git clone git://github.com/gozuk16/redmine_document_view_overwrite.git plugins/redmine_document_view_overwrite
```
- redmineの再起動

## Versions
- redmine 2.2.x
